# Davos Browser – A Fully Customisable, Privacy‑Focused Web Experience

Davos Browser is built for users who want complete control over their browsing environment.  
Every visual element can be customised — gradients, colours, animations, layout, and keybinds — all while maintaining a strong commitment to privacy and user autonomy.

Whether you prefer a minimal, distraction‑free interface or a vibrant, personalised aesthetic, Davos Browser adapts to your workflow and your style.

---

## Installation

Run the installer script:
```bash
chmod +x davosBrowserInstaller.sh && ./davosBrowserInstaller.sh
```
---

## Features

### Full Visual Customisation  
Davos Browser gives you complete control over its appearance:

- Custom gradients for UI elements  
- Adjustable colour schemes  
- Themeable interface components  
- Layout tweaks and visual presets  
- Optional animations and transitions  

Every part of the browser can be shaped to match your personal aesthetic.

### Custom Keybind System  
Designed for power users:

- Fully remappable keybinds  
- Modal navigation options  
- Workflow‑oriented shortcuts  
- Import/export keybind profiles  

Your browser adapts to you — not the other way around.

### Privacy First  
Davos Browser is built with privacy as a core principle:

- No telemetry  
- No background data collection  
- Privacy‑respecting defaults  
- Optional hardened configurations  

You stay in control of your data at all times.

### Performance‑Focused Architecture  
Lightweight, responsive, and optimised for modern systems:

- Fast startup  
- Efficient rendering pipeline  
- Minimal resource usage  
- Smooth animations and transitions
